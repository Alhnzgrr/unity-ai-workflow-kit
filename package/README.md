# Unity AI Workflow Kit Package

This folder is the Unity Package Manager entrypoint for **Unity AI Workflow Kit**.

Add it to a Unity project with:

```text
https://github.com/Alhnzgrr/unity-ai-workflow-kit.git?path=/package
```

Package contents:

- `AGENTS.md`
- `agents/`
- `commands/`
- `hooks/`
- `rules/`
- `skills/`
- `templates/`
- `workflows/`

Start by loading `AGENTS.md` into your AI tool context, then select the relevant role from `agents/`.
